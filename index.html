Meu Portfólio

Meu portfólio foi desenvolvido para apresentar meus projetos e habilidades como desenvolvedor, utilizando HTML, CSS, JavaScript, BootStrap, Git, Figma e Visual Studio Code. O objetivo principal foi criar um espaço profissional e interativo, demonstrando minha evolução e conhecimentos nas principais tecnologias de desenvolvimento web.

O projeto inclui uma estrutura organizada, com seções dedicadas as minha informações, dividas em: sobre mim, destaques, formações, projetos e habilidades, garantindo uma navegação intuitiva e visualmente atraente. Além disso, o código foi otimizado para responsividade, assegurando uma boa experiência em diferentes dispositivos. Essa iniciativa reforçou meu aprendizado em design, manipulação do DOM e boas práticas de desenvolvimento.

PetLife

O site PetLife foi desenvolvido como parte de um projeto introdutório na OneBitCode, utilizando HTML, CSS, JavaScript, Figma e Visual Studio Code. O objetivo principal era aprender os conceitos fundamentais dessas ferramentas de forma prática e introdutória na área de desenvolvimento web.

Jogo do Amigo Secreto

O jogo do Amigo Secreto foi desenvolvido como parte de uma atividade no Oracle Next Education, utilizando HTML, CSS, JavaScript, Git, Figma e Visual Studio Code. O objetivo principal era aplicar conceitos fundamentais dessas ferramentas de forma prática e interativa no ambiente de programação

MobiSeg
